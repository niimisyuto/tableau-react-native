# react-native 公式に tableau を追加した形

## react-native 導入手順

- git
- jdk 11.0.8
- androidstudio 33 tiramisu, adb, gradle 7.5.1 all 
- node 16.20 (nvm)
- watchman

## パス通す
- $env=JAVA_HOME = jdk11
- $env=ANDROID_HOME = android\sdk
- $env:Path = android\sdk\platform

## 設定

```shell
git clone
npm i
```

## 実行

### metroを起動 

```shell
npx react-native start
```
### bundle + android にコンパイル

```shell
a
```
### react-native debugモード

- android端末上でctrl + m
